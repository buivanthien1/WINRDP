# CREATE WINDOWS SERVER 
`HƯỚNG DẪN`

1.chọn `Ford`để mở ko lưu trữ

2.chọn `Actions`

3.chọn `Windows-SV-CRD` ở phần Workflows

4.chọn `Run Workflows`.Rổi `nhập Mã CRD google` và `Mật khẩu`

5.Sau đó chọn `Run Workflows`Để VPS được tạo (mất 2-3 phút)
